# 色づく世界

動物園を飛び出したパンダが買い物カートに乗り、マップ内を巡回するパトカーを避けながらカラーアイテムをゲットしていくゲームです。

# 動作環境
Windows7,8.1,10

# 開発環境
Unity(2020.3.8f1)

# 概要
三人称視点の3Dアクションゲームです。

# 操作方法
#### キーボード
Wキー:前進<br>
Sキー:ブレーキ<br>
Aキー:左旋回<br>
Dキー:右旋回<br>
Shiftキー:ブースト<br>
Spaceキー:急旋回(AorDキーと同時押し)<br>
<br>
#### マウス
左クリック:決定(メニュー操作時)<br>
